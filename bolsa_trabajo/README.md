# Bolsa de Trabajo - Equipo 3
Proyecto inicial en Django con MySQL para la cátedra de Ingeniería.

## Configuración Local
1. Clonar: `git clone <url-del-repo>`
2. Instalar dependencias: `pip install -r requirements.txt`
3. Crear base de datos en MySQL llamada `bolsa_db`.
4. Ejecutar: `python manage.py migrate`
5. Correr servidor: `python manage.py runserver`
